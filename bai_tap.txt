Đây là bài tập dành cho sửu nhuy 15 tháng 9
1. Sửa onTap & baiTap2 để in ra file thay vì ra màn hình
2. Viết code giải hệ phương trình 2,3,4 ẩn
3. Viết code tìm thứ của các ngày nhập vào từ file và in kết quả ra file
4. Viết code chuẩn hoá chuỗi
    - Viết hoa chữ cái đầu của câu, còn lại viết thường
    - Mỗi từ cách nhau đúng 1 đấu cách
    - Trước các dấu câu (.,?!) không có dấu cách nhưng sau thì có
    VD: sUu nHUY ,qUá      ? SaO đÂy    . --> Suu nhuy, quá? Sao đây.
